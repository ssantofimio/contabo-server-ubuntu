# Odoo 18 Custom Addons - SSantofimio

Repositorio dedicado a módulos personalizados y migraciones específicas para **Odoo 18**.

## Información General
- **Versión de Odoo:** 18.0
- **Propietario:** [SSantofimio](https://github.com/SSantofimio)
- **Estado:** Proyectos en versión 18.

## Contenido Destacado
- **Reportes Financieros:** `account_financial_report`, `accounting_pdf_reports` (v18).
- **POS (Punto de Venta):** `bi_pos_combo`, `pos_full_product_name`, `pos_product_schedule`.
- **Gestión de Inventario:** `sandor_it_inventory`, `stock_picking_invoice_link`.
- **Migración:** Contiene herramientas y logs relacionados con el proceso de actualización a Odoo 18 (`migration_tool`).

## Configuración
Para utilizar estos módulos, agrégalos a tu `addons_path` en Odoo 18 y reinicia el servicio antes de buscarlos en la interfaz de aplicaciones.
