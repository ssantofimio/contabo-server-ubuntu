Display Price On Product Screen of POS

Description

Price Display for Odoo (POS) Product Cards

This module enhances the Odoo (POS) interface by adding price to product cards. Each product card now displays the list price, making it easier for cashiers and customers to quickly identify product prices during the ordering process.
