# Sarupetrol Custom Addons - Odoo 18

Este repositorio contiene la personalización específica para el proyecto **Sarupetrol**, basada en **Odoo 18**.

## Información del Proyecto
- **Cliente/Contexto:** Sarupetrol
- **Versión Base:** Odoo 18.0
- **Enfoque:** Módulos de contabilidad, inventario y reportes adaptados a las necesidades de Sarupetrol.

## Estructura de Módulos
Módulos clave incluidos:
- `custom_pnl_report`: Reporte de Pérdidas y Ganancias personalizado.
- `base_accounting_kit`: Suite contable adaptada.
- `sandor_pos_self_order_ses`: Personalización para auto-pedido en POS.
- `sttl_sale_subscription`: Gestión de suscripciones y ventas.

## Uso
Este repositorio es específico para la instancia de Sarupetrol. Al instalar, verificar las dependencias de terceros incluidas en la carpeta de origen.
