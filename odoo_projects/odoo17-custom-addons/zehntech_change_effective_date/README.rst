Change Effective Date
================================================================

The **Change Effective Date** module enables users to modify effective dates on key transactions such as invoices, sales orders, purchase orders, and stock operations. It supports backdating and future-dating while ensuring accurate currency rate application and inventory valuations. With role-based access control, it provides secure and flexible date management, maintaining data integrity across financial and inventory reports.

**Key Features**
================================================================

- **Change the Effective Date of Transactions Easily**:Allows users to update the effective date of confirmed transactions like invoices, sales, and purchase orders.Admins can control who can change dates using user-specific privileges.
- **Apply Currency Rates Based on the Selected Date**: Applies accurate currency exchange rates according to the manually selected effective date. Ensures proper valuation in multi-currency transactions and financial reports.
- **Update Invoices, Sales, and Purchase Orders as per the Effective Date Values**: Enables backdating or correcting effective dates on key documents post-confirmation. Valuations and reports reflect the updated dates, ensuring consistency and audit accuracy. 
 
**Summary**
================================================================

Manage and adjust effective dates for sales, purchase, and inventory records. Change Effective Date Odoo module enables accurate historical data representation for currency rates, stock valuation, and financial reporting. Features configurable access control for data integrity

**Installation**
================================================================

1. Download the module from the Odoo App Store or clone the repository.
2. Place the module in your Odoo addons directory.
3. Update your Odoo instance to include the new module.
4. Install the module through the Odoo interface.

**How to use this module:**
================================================================
1. Navigate to any Sales Order, Purchase Order, or stock Receipt form.
2. Locate the "Effective Date" field (editable for authorized users).
3. Select or edit the date and apply changes using the smart button.
4. For validated documents, only users with the "Change Effective Date" privilege can edit the date.
5. Changes reflect immediately in inventory valuation and reports.

**User Access Setup**
================================================================

1. Navigate to: **Settings → Users & Companies → Users**
2. Select the user and enable **Change Effective Date Privilege**
3. Save changes. The user can now edit effective dates where permitted.
