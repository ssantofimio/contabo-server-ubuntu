Restrict Pickings and Stock Moves Delete

Description

This module is configured to restrict the deletion option for pickings and stock moves using a group, enhancing control and maintaining data integrity in the inventory management process.

Configuration

After installing the Stock Restrict Delete application, the feature will be enabled.

Go to Settings - - > Users & Companies - - > Users.

Select the user where the “Can Delete DO” boolean group is added under the Access Rights Tab.

Functionality

When the “Can Delete DO” boolean is enabled, it restricts the delete button in the list and form view for the stock move and picking.

After activation of the “Can Delete DO” boolean on the user access.

The Delete action is visible on the Stock Move list view

The Delete action is visible on the Pickings list and form view

After False of the “Can Delete DO” boolean.

The Delete action is invisible on the Stock Move list view

The Delete action is invisible on the Pickings list and form view
