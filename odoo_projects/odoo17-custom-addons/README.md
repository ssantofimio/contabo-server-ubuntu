# Odoo 17 Custom Addons - SSantofimio

Este repositorio contiene módulos personalizados desarrollados y adaptados para **Odoo 17 Community/Enterprise**.

## Información General
- **Versión de Odoo:** 17.0
- **Propietario:** [SSantofimio](https://github.com/SSantofimio)
- **Descripción:** Colección de herramientas, reportes y funcionalidades específicas para la gestión empresarial en Odoo 17.

## Módulos Incluidos (Resumen)
Este repositorio incluye, entre otros, los siguientes componentes:
- **Contabilidad y Reportes:** `accounting_pdf_reports`, `om_account_accountant`, `om_fiscal_year`.
- **Inventario y Stock:** `inventory_advanced_reports`, `stock_negative_balance_preventer`, `sandor_it_inventory`.
- **Dashboards:** `odash_pro`, `odoo_dynamic_dashboard`, `synconics_bi_dashboard`.
- **Utilidades:** `ica_web_responsive`, `login_as_any_user`, `dms` (Document Management System).

## Instalación
1. Clonar este repositorio dentro de la carpeta de `addons` de tu instancia de Odoo 17.
2. Asegurarse de que la ruta esté incluida en el `addons_path` del archivo de configuración de Odoo.
3. Actualizar la lista de aplicaciones e instalar el módulo deseado.
